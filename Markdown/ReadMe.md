# Markdown-Spickzettel

## Eine Kurzübersicht über die Markdown-Syntax.

### Übersicht

Dieses Markdown-Spickzettel bietet einen schnellen Überblick über alle Markdown-Syntaxelemente. Es kann nicht alle Sonderfälle abdecken. Wenn Sie weitere Informationen zu einem dieser Elemente benötigen, lesen Sie bitte die Referenzhandbücher zur grundlegenden Syntax und zur erweiterten Syntax.

### Grundlegende Syntax

Dies sind die Elemente, die in John Grubers ursprünglichem Designdokument beschrieben sind. Alle Markdown-Anwendungen unterstützen diese Elemente.
